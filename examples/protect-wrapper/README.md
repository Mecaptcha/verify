# Protect Wrapper Example

Example showing the `<MeCaptchaProtect />` wrapper component that hides content until verified.

## Run

```bash
cd examples/protect-wrapper
pnpm install
pnpm dev
```

Open http://localhost:5173

## Features

- Automatic verification flow
- Protected content only shown after verification
- Session persistence (survives page refresh)
- Context API for accessing verification state
- Sign out functionality
